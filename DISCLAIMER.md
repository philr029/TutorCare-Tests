# Legal & Ethical Usage Disclaimer

## Intended Use
This Security Diagnostic Toolkit is designed exclusively for:
- Authorised security research and analysis
- System and network administration by qualified professionals
- Diagnosing issues with systems you own or have explicit written permission to test
- Educational purposes in controlled environments

## Prohibited Uses
**You must NOT use this toolkit to:**
- Test, probe, or analyse systems without explicit written authorisation
- Attempt to bypass, circumvent, or defeat security controls of any kind
- Engage in any form of unauthorized computer access (illegal in most jurisdictions)
- Harass or target individuals via phone number lookups
- Violate the terms of service of any third-party API used by this toolkit

## Legal Notice
Unauthorized use of network diagnostic tools may violate:
- The Computer Fraud and Abuse Act (CFAA) – United States
- The Computer Misuse Act 1990 – United Kingdom
- Similar legislation in your jurisdiction

The authors and contributors assume **no liability** for misuse of this toolkit.
By using this software, you agree to comply with all applicable laws and regulations.

## Responsible Disclosure
If you discover a vulnerability using this toolkit, please follow responsible disclosure
practices and report it to the affected organisation before public disclosure.

## Data Privacy
- API keys are redacted from logs
- Phone number lookups are non-intrusive (no calls, pings, or probes)
- Website checks use a clearly identified User-Agent string
- No data collected by this tool is shared with third parties by default
